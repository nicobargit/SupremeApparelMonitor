# Supreme Apparel Monitor
Python script based on selenium webdriver to email notify you when a product became in-stock<br>
To install requirements run:
```
pip install -r requirements.txt
```
Launch the following command for details:
```
python supreme_monitor.py -h
```
Usage:
```
python supreme_monitor.py -url <supreme_item_url> -gmail-from <gmail_mail_sending_notify> -gmail-password <gmail_account_password> -mail-to <mail_to_get_notified> -sleep-time <sleep_time_between_checks>
```
